Налаштування Firebase

Firebase — безкоштовний хмарний сервіс Google для синхронізації між телефонами в реальному часі.

  1. Відкрий console.firebase.google.com
  2. Створи новий проект (будь-яка назва)
  3. Перейди до Realtime Database → Створити базу даних
  4. Обери «Тестовий режим» (Test mode)
  5. Скопіюй URL бази — виглядає як:
    https://ім'я-default-rtdb.firebaseio.com
  6. Встав URL нижче
